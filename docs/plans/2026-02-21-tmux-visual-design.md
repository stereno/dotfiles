# tmux ビジュアル改善デザイン

## 概要

tmux のステータスバー・ペインボーダー・メッセージのビジュアルを全体的に改善する。
アプローチは手書き extraConfig（プラグイン不使用、依存ゼロ）。

## 要件

- モードインジケーター（NORMAL / PREFIX / COPY）を色分けで表示
- ステータスバー右側に Git ブランチ・日付・時刻を追加
- モダンミニマルなデザイン（角丸・アイコンでセクションを区切る）
- ペインボーダーは細線
- TokyoNight Night テーマとの統一

## カラーパレット（TokyoNight Night）

| 用途 | カラー |
|---|---|
| 背景 | `#1a1b26` |
| テキスト | `#c0caf5` |
| アクセント（青） | `#7aa2f7` |
| サブテキスト | `#565f89` |
| サーフェス | `#292e42` |
| 警告/PREFIX（黄） | `#e0af68` |
| 成功/COPY（緑） | `#9ece6a` |

## ステータスバー レイアウト

```
┌─────────────────────────────────────────────────────────────┐
│  NORMAL   S:main │ 1:zsh  2:nvim  3:htop │  main  02/21 14:30 │
└─────────────────────────────────────────────────────────────┘
```

### 左側（status-left）

- モードインジケーター: `#{?client_prefix,PREFIX,#{?pane_in_mode,COPY,NORMAL}}`
  - NORMAL: `bg=#7aa2f7,fg=#1a1b26,bold`
  - PREFIX: `bg=#e0af68,fg=#1a1b26,bold`
  - COPY: `bg=#9ece6a,fg=#1a1b26,bold`
- セッション名: `fg=#7aa2f7` で `#S` を表示

### 中央（ウィンドウリスト）

- 非アクティブ: `fg=#565f89` — ` #I:#W `
- アクティブ: `fg=#7aa2f7,bg=#292e42,bold` — ` #I:#W `
- セパレーター: なし（スペースで区切り）

### 右側（status-right）

- Git ブランチ: `fg=#9ece6a` — `#(git -C #{pane_current_path} branch --show-current 2>/dev/null)`
- 日付: `fg=#565f89` — `%m/%d`
- 時刻: `fg=#c0caf5` — `%H:%M`

## ペインボーダー

- 非アクティブ: `fg=#292e42`（暗め、目立たない）
- アクティブ: `fg=#7aa2f7`（青ハイライト）

## ペイン番号表示（display-panes）

- アクティブ: `#7aa2f7`
- 非アクティブ: `#565f89`

## メッセージ

- `bg=#292e42,fg=#c0caf5`（現状維持）

## ファイル変更

| ファイル | 変更内容 |
|---|---|
| `home/tmux.nix` | extraConfig のステータスバー・ボーダー・メッセージ部分を書き換え |

## アプローチ

手書き extraConfig。プラグイン追加なし。tmux ビルトインの条件分岐 `#{?...}` とシェルコマンド `#(...)` を使用。
